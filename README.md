# Лабораторные работы 1-4
## Дисциплина: «Моделирование и надежность систем»
### Вариант 14: Информационная система интернет-магазина (консоль)

### Лабораторная работа 1: Принципы SOLID (ветка lab1)
Разработка базового консольного приложения с фокусом на соблюдение принципов проектирования SOLID, DRY и KISS. Заложена базовая структура сущностей и реализована иерархия доменных исключений.

### Лабораторная работа 2: Паттерны проектирования (ветка lab2)
Рефакторинг системы с внедрением классических паттернов проектирования. Использованы паттерны Strategy, Factory и Observer для повышения гибкости кодовой базы и упрощения добавления нового функционала.

### Лабораторная работа 3: Антипаттерны (ветка lab3)
Практическое изучение влияния технического долга. Расширение системы новым модулем со сложной логикой расчета стоимости, который намеренно реализован с использованием антипаттернов (God Object, Magic Numbers) в условиях изоляции от чистой кодовой базы.

### Лабораторная работа 4: Базовый системный дизайн (ветка lab4)
Декомпозиция монолита на независимые микросервисы (Core Service и Reference Service). Настроено межсервисное взаимодействие по протоколу gRPC, добавлено сквозное логирование запросов через Trace ID и реализована стратегия graceful degradation при сетевых сбоях.
